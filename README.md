# BD-FIB

Coursework repository for **BD (Bases de Datos)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository contains practical database work, exercises, and solutions related to the course.

## Overview

BD introduces core database concepts, including modeling, querying, and working with relational systems in a practical way.

This repo collects the material developed during the course and keeps it organized as a study and reference resource.

## Contents

- Practical assignments
- SQL exercises
- Database-related scripts

## Tech

- SQL
- PostgreSQL / PLpgSQL
- Relational database concepts

## Notes

This is an academic repository for the BD coursework at FIB.

Its main purpose is to document and preserve the practical work completed throughout the subject.
